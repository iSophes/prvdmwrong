# getIgnitionOptions

Returns the startup configuration options. Options are reconciled with
default values.

=== "Luau"

    ```Lua
    function prvd.getIgnitionOptions(): Options
    ```

=== "TypeScript"

    ```TypeScript
    export const getIgnitionOptions: () => Options
    ```

---

## Returns

The startup options, reconciled with default values.
